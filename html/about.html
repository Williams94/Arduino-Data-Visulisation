Arduino Data Visualization

Description of Project

This project visualizes data captured by an Arduino Uno.

There are several sensors used to capture data about the current environment:

  • Photoresistor
  • Temperature Sensor
  • Sound Sensor
  • Vibration Sensor

The data the Arduino receives from these sensors is sent over serial connection to the server (written in Node.js), which then stores them into the database (MongoDB). The server also handles all of the requests for the front end web application (Bootstrap), with the data visualization created using D3.js. Web Sockets are used to retrieve the sensor data from the database and use them client-side with the D3.js library for the data visualization.
